## 🚀 Getting Started

### 1. 프로젝트 클론

```bash
git clone https://github.com/boostcampwm2025/web21-funda.git
```

### 2. 의존성 설치 및 실행

> pnpm을 사용하여 모노레포를 관리합니다.
> 

```bash
# 의존성 설치 (루트 디렉토리에서)
pnpm install

# 개발 환경 실행
pnpm run dev
```
